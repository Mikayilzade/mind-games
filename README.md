# Mind Games

Мобильный тренажёр памяти, внимания и логики с ежедневным планом, возвращением знакомых задач и понятной историей личного прогресса.

## Версия 1.3.0

- четыре активных режима: «Вспышка», «Пары», «Цепочка» и полноценное судоку 9×9;
- игровой экран помещается в доступную высоту на телефоне и компьютере без внутренней прокрутки;
- двойной тап на игровых элементах не должен масштабировать страницу;
- спокойная светлая тема используется по умолчанию, тёмная доступна в профиле;
- судоку поддерживает карандашные пометки, ввод 1–9 с клавиатуры, стрелки, Backspace/Delete, `P` и `Ctrl/Cmd+Z`;
- при постоянном вводе цифры заметки в связанных клетках обновляются автоматически;
- видимые конфликты подсвечиваются, а отмена ошибочного хода восстанавливает предыдущие заметки, исключая уже проверенный неверный кандидат;
- автоматический список возможных цифр убран;
- подсказки двухступенчатые: сначала логический намёк, затем по отдельному запросу конкретное число;
- каждое задание имеет постоянный скрытый идентификатор и может вернуться через 5–10 дней в неизменном виде;
- повтор сравнивается с прошлым прохождением по времени, точности и ошибкам;
- для судоку сохраняется журнал действий как основа будущего реплея;
- история, опыт, уровни, серии дней и достижения хранятся локально без регистрации;
- шахматный режим остаётся архивным до полной отдельной переработки.

## Запуск локально

Приложение не требует сборки или установки зависимостей.

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Проверки

Локально проверены:

- синтаксис приложения, загрузчика и service worker;
- 245 судоку семи уровней — корректное единственное решение;
- рендер всех четырёх игр на экранах 1440×900 и 390×844 без переполнения игрового окна;
- клавиатурная навигация судоку, карандаш, конфликт и специальная отмена хода.

В репозитории намеренно нет GitHub Actions: локальные проверки не создают уведомлений по электронной почте.
