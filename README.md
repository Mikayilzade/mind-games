# Mind Games

Мобильный тренажёр памяти, внимания и логики с ежедневным планом и понятной историей личного прогресса.

## Что уже работает

- ежедневный план из трёх адаптивных заданий;
- пять полноценных режимов: «Вспышка», «Пары», «Цепочка», мини-судоку и шахматные мотивы;
- автоматическое возвращение старых заданий через несколько дней для проверки долговременной памяти;
- сравнение повторного результата с предыдущим по времени, точности и баллам;
- опыт, уровни, серии дней и достижения;
- недельная активность и история результатов;
- локальное хранение данных без регистрации;
- установка на домашний экран iPhone и офлайн-работа после первого запуска.

## Запуск локально

Приложение не требует сборки или установки зависимостей.

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Публикация

Проект полностью статический и подходит для GitHub Pages. В настройках репозитория выберите **Settings → Pages → Deploy from a branch → `agent/mind-games-v1` / root** для теста ветки либо `main` после слияния PR.

## Проверки

В репозитории намеренно нет GitHub Actions: локальные проверки не создают уведомлений по электронной почте.
