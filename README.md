# Mind Games

Мобильный тренажёр памяти, внимания и логики с ежедневным планом и понятной историей личного прогресса.

## Что уже работает

- ежедневный план из трёх адаптивных заданий;
- пять режимов: «Вспышка», «Пары», «Цепочка», судоку и шахматные задачи;
- судоку развивается от обучающего 4×4 к 6×6 и полноценному 9×9;
- строки, колонки и блоки судоку визуально выделяются, конфликты объясняются;
- шахматные задачи содержат понятную цель, координаты, читаемые фигуры и объяснение решения;
- через 30 секунд в судоку и шахматах становится доступна подсказка;
- старые задания автоматически возвращаются для проверки долговременной памяти;
- повторный результат сравнивается с предыдущим по времени, точности и баллам;
- опыт, уровни, серии дней, достижения и история результатов;
- локальное хранение данных без регистрации;
- установка на домашний экран iPhone и офлайн-работа после первого запуска.

## Запуск локально

Приложение не требует сборки или установки зависимостей.

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Публикация

Проект полностью статический и публикуется через GitHub Pages из ветки `main`, каталог `/root`.

## Проверки

В репозитории намеренно нет GitHub Actions: локальные проверки не создают уведомлений по электронной почте.
